# 🌌 Infinite Galaxy Generator

An interactive 3D spiral galaxy generator built with Three.js. Each click generates a completely unique galaxy with random arm counts, colors, spin, and density.

## 🚀 Live Demo
[Click here to experience it live!](https://your-username.github.io/galaxy-generator)

## ✨ Features
- **Procedurally Generated** – Every galaxy is algorithmically created with random parameters.
- **Interactive** – Move your mouse to orbit around the galaxy; click to spawn a brand-new one.
- **Glowing Particles** – Thousands of colored particles with additive blending and soft falloff textures.
- **Smooth Animations** – Auto-rotation with mouse-driven interpolation for a buttery feel.
- **Unique Color Palettes** – Warm cores (yellow/orange) blending into cool edges (blue/purple/pink).
- **Pure Three.js** – Built with the latest Three.js (ES module) loaded via CDN.

## 🛠️ Built With
- **Three.js** – 3D rendering engine.
- **Vanilla JavaScript (ES Modules)** – All logic is in a single module script.
- **HTML5 Canvas** – For generating the soft particle texture on the fly.
